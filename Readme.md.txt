# 🎓 EduInsight – Student Performance Dashboard

## 📊 Overview
This project is an interactive Power BI dashboard built to analyze student performance data and identify key factors affecting scores.

---

## 🚀 Key Features
- 📈 Average Score visualization (Gauge Chart)  
- 👨‍🎓 Gender-wise performance comparison  
- 🎓 Parental education impact analysis  
- 🍱 Lunch type vs performance  
- 📘 Test preparation course insights (Donut Chart)  
- 🎛️ Interactive slicers (Gender filter)  
- 📋 Clean data table for detailed view  

---

## ✅ Tasks Completed
- ✔ Cleaned dataset using Python (Pandas)
